生前契約常見問題

生前契約是什麼？為什麼需要？

生前契約是指在生前就與禮儀公司簽訂協議，預先規劃和支付未來的喪禮費用。主要優勢包括：

  • ✓ 鎖定現價 - 不受通膨影響
  • ✓ 減輕家屬負擔 - 親人不需倉皇決定
  • ✓ 自主規劃 - 按自己意願安排
  • ✓ 法律保護 - 75% 信託保障

什麼時候該簽生前契約？

沒有絕對的「最好時機」，但建議在以下情況簽訂：

  • • 55～70 歲 - 身體健康、意識清楚的年紀
  • • 退休前後 - 經濟穩定、有時間思考
  • • 親友過世後 - 體會到完善規劃的重要性
  • • 任何想為家人減負擔的時刻

信託的 75% 是什麼意思？25% 呢？

依據內政部規定，生前契約款項中：

💰 75% - 信託專戶保護

存入指定銀行信託帳戶，由第三方金融機構監管，只能用於契約服務。

📋 25% - 業者維運費用

用於業者營運、維護會館、管理成本等，無信託保護。

簽訂後可以退費嗎？

✓ 可以退費，條件如下：

簽約未滿 2 年 退 95% 信託款 + 利息
滿 2 年以上 退 100% 信託款 + 利息

※ 25% 維運費用依實際情況評估返還

契約可以轉讓嗎？

是的，契約可以轉讓。

如果簽約人有以下情況，可將契約轉讓給其他人（配偶、子女等）：

  • • 改變主意，不想使用該服務
  • • 搬遷到外地
  • • 想由其他家屬代理

✓ 轉讓程序簡單，只需向華梵申請變更契約人名義

華梵禮儀是合法業者嗎？

✓ 是的，華梵禮儀完全合法。

  • 內政部核准 - 生前契約服務業者認可
  • 定型化契約 - 符合消保法規定
  • 信託銀行 - 與合法金融機構合作
  • 履約保證金 - 提撥至內政部
